# dingus Fastfetch
## Setup
Copy `config.jsonc` to `~/.config/fastfetch`\
That's basically it\
## Images
By default, fastfetch will use an ASCII version of your distro logo\
You can add an image by renaming it to `logo.png` and copying it to `~/.config/fastfetch`\
250x250 is usually the best size for these images
